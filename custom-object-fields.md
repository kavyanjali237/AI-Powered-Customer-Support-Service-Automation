# Custom Object Fields

## Customer__c Object

| Field Label     | API Name            | Data Type | Values |
|----------------|---------------------|----------|--------|
| Customer Name  | Customer_Name__c    | Text     | - |
| Email          | Email__c            | Email    | - |
| Phone          | Phone__c            | Phone    | - |
| Customer Type  | Customer_Type__c    | Picklist | Individual, Business |

## Steps Performed

1. Open Salesforce Setup.
2. Go to Object Manager.
3. Create a Custom Object named Customer.
4. Create the following fields:
   - Customer Name (Text)
   - Email (Email)
   - Phone (Phone)
   - Customer Type (Picklist)
5. Save the object.

## Result

The Customer__c object was successfully created with all required custom fields.
