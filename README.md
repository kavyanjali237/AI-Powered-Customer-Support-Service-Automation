# AI-Powered-Customer-Support-Service-Automation
## Project Overview
This project demonstrates stakeholder mapping for an AI-powered customer support and service automation solution using Salesforce Agentforce and Prompt Builder.

## Objectives
- Identify project stakeholders.
- Define stakeholder roles and responsibilities.
- Explain stakeholder interactions.
- Improve communication and project success.

## Stakeholders
- Project Sponsor
- Salesforce System Administrator
- Inspection Team
- Support Team
- Management
- IT Support

## Technologies
- Salesforce
- Agentforce
- Prompt Builder
- GitHub
